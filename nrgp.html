Non Returnable Gate Pass(RGP)

  • Inventory Management >Periodic Tasks > Gate Management > All Outword Gate Entries
  • Select New(Outward gate entry)


  • In header section fill all Required details Factory Gate, warehouse , location,Refernece Document Type ,challan Number, Challan Date and NRGP Number than save.

    • Note:- Need to specify NRGP Number sequence Statrted with store code than after with any number like PUNE-FCRD01.

    Note: The Reference Party field is not applicable when the Reference Document Type field is selected as Others.

  • After Entering All the details , Save the data , Now click the Add line.
  • After Clicking the add line, select the Item which is not return back to the store.
  • Save and click the confirm vehicle Entry.
  • After this go to show Gate entry where the all data Gate Number, vendor name will shown, we can Export the file as Excel OR PDF.
  • Next go to confirm vehicle Exist.
  • Now the Non-Retune process has been completed
  • To Check the all the RGP & NRGP data , Go to the RGP Report.

    • Inventory Management>Enquiries and reports>RGP Report

  • Select the from date,To Date and Store Code than click Ok.
  • RGP report will be shown ,where we will seen all gate entry type, store name , Item Quantity .we Export the report in any format .
