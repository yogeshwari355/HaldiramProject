Indent Creation

  • Retail & Commerce >Channels > Store >Indent
  • Select New


  • After click on new button new indent number will generate and userâ€™s store & warehouse details will come automatically based on the user setup.
  • User should select the indent to warehouse & shift code, demand date will come automatically.
  • Now user can add the product which is under the same coverage group.

  • After adding all the item user should click on the ok button all the selected items will add into the indent line.
  • Into the indent line user should enter the required qty or user use the system suggested qty.
  • Into the indent line user can change the unit (Only the available conversion units will show into the list of value.) as per the user requirement.
  • After completing the Qty update process user should client on the POST button.
  • After click on the POST button system will generate the document based on the indent document type (Purchase Order)and category name.

    • GRN Process

  • For GRN Go to Procurement and sourcing > Purchase orders >All purchase orders.
  • Select the created Purchase Order
  • In the paticular Purchase order, In purchase tab go the action and click confirm .
  • After Confrimation, In receive tab go to the Product Receipt
  • Click the sale order Invoice and select the sale order number which will present on invoice.
  • If the Item receive short or damage than go the Upadate Line select Reservation.
  • Split the Quantity according the received Quantity.
  • Click on add Resistration line.
  • In Registrations Line select location , click on confirm location and Save.
  • Click Ok in product Receipt, Now the GRN process has been completed.
